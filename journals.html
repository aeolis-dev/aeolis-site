Vutu to Zebra 3 Modal Profile Converter
Zebra 3 is a new software synthesizer currently in open beta. It has a "modal" module which allows the user to synthesize the timbres of real resonant objects using modal synthesis. There is a lack of factory profiles for this, so I made a web app to convert .utu files to Zebra 3 .csv profiles. .utu files are created by Vutu, a free program for analysing the spectral profiles of audio sources.
