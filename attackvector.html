AttackVector Details
AttackVector is a 2D arcade shooter developed in Unreal Engine 5 using C++ and Unreal Engine's blueprint system. The gameplay and visual style is a retro-futuristic throwback to arcade games of the 80s.

Personal Contributions
For the final version, my primary contributions were major work on the game's core mechanics, and logic related to physical game assets. I had also written a large C++ header and base class for the "enemy" assets at the first stage of development, but the team had opted to exclusively use Unreal Engine Blueprints as opposed to a code-based approach. I spent a lot of time walking the team through effective GitHub use and making sure issues were tracked and resolved.

An extensive list of my contributions as follows:

- All logic for the "cross" enemy, and the boss version which used more intelligent movement patterns.

- All logic for the visual hit feedback, requiring dynamic materials attached to the all enemies' meshes.

- Refactoring of certain enemies to a boss class, requiring a large amount of logic to be rewritten.

- Rewriting the "Hexagon" boss logic, which required an additional custom material for telegraphing its warp, and adjustments to the spinning/shooting logic to work more consistently across machines.

- All materials had to be refactored to allow for dynamic mesh spawning.

- Various bug fixes across the project, including changes to levels, materials, projectiles and enemies assigned to other team members.
